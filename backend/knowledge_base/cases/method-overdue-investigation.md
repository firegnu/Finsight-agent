# 方法论 — 逾期率异常的标准调查流程

## 适用场景

月度逾期率超过告警阈值（通常为基线 +1 个标准差 / 或绝对值 >5%）时，按本 SOP 定位根因、避免漏判与误判。

## Step 1：形态识别（决定后续调查方向）

**骤升型**（3 个月内翻倍 / 显著偏离其他区域）：
- 首查"审批策略 / 风控规则"近 3-6 个月是否有变动（滞后效应明显）
- 参考案例：east-2024-q3-overdue-spike（审批分数下调）

**缓升型**（连续 2-3 个月小幅上升）：
- 首查"区域宏观经济 / 客群结构"变动
- 参考案例：central-2025-q1-overdue-climb（制造业裁员）

**骤变型**（单月跳点）：
- 首查"系统故障 / 账务异常 / 特殊客群集中到期"
- 看清是不是 M1/M2/M3 分布异常（而非整体逾期率异常）

## Step 2：客群拆层（定位受影响群体）

按以下维度拆分逾期率，任一维度内部差异显著即是根因候选：
- **信用评分段**：680- / 680-720 / 720-760 / 760+
- **客户来源渠道**：地推 / 线上 / 转介绍 / 存量升级
- **职业分类**：蓝领 / 白领 / 个体 / 公务员
- **入卡时长**：<6m / 6-12m / 12-24m / >24m
- **M1 滚存率**（先行指标，早 1-2 个月预警逾期率变化）

## Step 3：时间溯源（定位因果起点）

- **审批策略溯源**：近 6 个月审批评分卡、决策规则、放款权限是否变动？
- **营销活动溯源**：近 3 个月是否有大规模低门槛营销（免息分期 / 首刷礼）？低意愿客群混入是主因之一
- **宏观因子溯源**：区域失业率、PMI、主导产业景气度

## Step 4：交叉验证（排除误判）

- 同期同业数据（若可获）：全行业上行 → 宏观因子；我行独升 → 内生问题
- 客户投诉文本分析："逾期"相关投诉是否有系统扣款失败、账单错算等技术原因
- 催收回收率同步变化：同向下滑说明"新增逾期 + 存量回收"双杀（最坏情况）

## Step 5：根因收敛与应对

**常见根因分类**：
1. 审批门槛放宽 → 立即回调 + 存量客户专项（最快见效）
2. 宏观经济承压 → 专项减损 + 客群结构调整（长期作战）
3. 系统/账务异常 → 紧急修复 + 误伤客户补偿（时效第一）
4. 催收能力下降 → 扩容内部 + 新增外包（中期）
5. 营销策略副作用 → 缩减低门槛活动 + 引入 ARPU 考核权重

**避免的错误**：
- 只看月度均值不拆客群 → 混合问题被掩盖
- 忽视 3-6 个月滞后效应 → 看短期"策略有效"而放任根因
- 单一根因思维 → 多数事件是 2-3 个因素叠加
